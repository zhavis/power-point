# دسته‌بندی کلی کابل‌های شبکه

## انواع اصلی کابل‌ها بر اساس رسانه انتقال

کابل‌های شبکه را می‌توان از زوایای مختلف دسته‌بندی کرد. مهم‌ترین دسته‌بندی بر اساس نوع رسانه انتقال است:

### ۱. **کابل‌های مسی (Copper Cables)**
- **زوج‌به‌هم تابیده (Twisted Pair):** شامل کابل‌های Cat5e، Cat6، Cat6a، Cat7، Cat8
- **کواکسیال (Coaxial):** کابل‌های قدیمی (مانند RG-58، RG-59، RG-6)
- **کابل‌های تخت (Flat Cables):** کاربردهای خاص

### ۲. **کابل‌های فیبر نوری (Fiber Optic Cables)**
- **تک‌حالته (Single-mode - SMF):** برای مسافت‌های طولانی
- **چندحالته (Multi-mode - MMF):** برای مسافت‌های کوتاه‌تر
- **فیبر پلاستیکی (POF):** کاربردهای خاص

### ۳. **کابل‌های بی‌سیم (Wireless)**
- هرچند کابل نیست، اما به عنوان رسانه انتقال محسوب می‌شود
- امواج رادیویی (Wi-Fi، بلوتوث، 5G)
- مادون قرمز (Infrared)

# بخش اول: کابل‌های مسی زوج‌به‌هم تابیده

## پرکاربردترین کابل‌های شبکه

کابل‌های زوج‌به‌هم تابیده (Twisted Pair) رایج‌ترین نوع کابل‌های شبکه هستند که در اکثر شبکه‌های محلی (LAN) مورد استفاده قرار می‌گیرند. این کابل‌ها از چهار جفت سیم مسی تشکیل شده‌اند که هر جفت به دور یکدیگر تابیده شده‌اند تا از تداخل الکترومغناطیسی (EMI) جلوگیری شود.

### ساختار کابل زوج‌به‌هم تابیده

```
┌─────────────────────────────────────────────────────────┐
│  🧵 روکش خارجی (Outer Jacket) - PVC/LSZH/Plenum       │
│  ┌───────────────────────────────────────────────────┐  │
│  │  🛡️ شیلد (Shield) - اختیاری (FTP/STP/SFTP)      │  │
│  │  ┌─────────────────────────────────────────────┐  │  │
│  │  │  🧵 غلاف داخلی (Inner Sheath)              │  │  │
│  │  │  ┌───────┐  ┌───────┐  ┌───────┐  ┌───────┐│  │  │
│  │  │  │ ════╗│  │ ════╗│  │ ════╗│  │ ════╗││  │  │
│  │  │  │ ════╝│  │ ════╝│  │ ════╝│  │ ════╝││  │  │
│  │  │  │ جفت ۱ │  │ جفت ۲ │  │ جفت ۳ │  │ جفت ۴ ││  │  │
│  │  │  └───────┘  └───────┘  └───────┘  └───────┘│  │  │
│  │  └─────────────────────────────────────────────┘  │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

### اجزای کابل زوج‌به‌هم تابیده:

| جزء | توضیحات |
|-----|---------|
| **هادی (Conductor)** | سیم مسی با خلوص بالا (مس خالص یا مس آلومینیومی) |
| **عایق (Insulation)** | پوشش پلاستیکی دور هر سیم (PE/PP) |
| **جفت‌های تابیده (Twisted Pairs)** | ۴ جفت سیم با گام‌های تابش متفاوت |
| **شیلد (Shield)** | لایه محافظ در برابر تداخل (در کابل‌های Shielded) |
| **غلاف داخلی (Inner Sheath)** | لایه جداکننده بین جفت‌ها |
| **روکش خارجی (Outer Jacket)** | پوشش نهایی محافظ (PVC/LSZH/Plenum) |

## بررسی دقیق هر استاندارد

### 📌 **Cat5e (Category 5 Enhanced)**

**توضیحات:** نسخه بهبودیافته Cat5 که همچنان یکی از پرکاربردترین کابل‌ها برای شبکه‌های خانگی و اداری کوچک است.

**ویژگی‌ها:**
- پشتیبانی از سرعت ۱ گیگابیت بر ثانیه تا ۱۰۰ متر
- قیمت بسیار مناسب
- موجود و در دسترس
- مناسب برای اکثر کاربردهای روزمره

**کاربردها:**
- شبکه‌های خانگی
- دفاتر کوچک (کمتر از ۵۰ کاربر)
- اتصال دستگاه‌های معمولی (کامپیوتر، چاپگر)
- شبکه‌های دوربین مداربسته (با سرعت پایین)

**مزایا:** قیمت پایین، در دسترس بودن، سهولت نصب

**معایب:** پهنای باند محدود، عدم پشتیبانی از ۱۰ گیگابیت

**قیمت تقریبی:** $ (ارزانترین)

---

### 📌 **Cat6 (Category 6)**

**توضیحات:** استاندارد بهبودیافته با عملکرد بهتر و پهنای باند بالاتر نسبت به Cat5e.

**ویژگی‌ها:**
- پشتیبانی از ۱۰ گیگابیت تا ۵۵ متر
- کاهش تداخل (Crosstalk) با جداکننده‌ها
- پهنای باند ۲۵۰ مگاهرتز
- معمولاً دارای جداکننده پلاستیکی (Spline) بین جفت‌ها

**کاربردها:**
- شبکه‌های سازمانی متوسط
- دیتاسنترهای کوچک
- شبکه‌های با ترافیک بالا
- اتصال سرورها در یک رک

**مزایا:** سرعت بالا در فواصل کوتاه، قیمت مناسب

**معایب:** افت سرعت در فواصل بیشتر از ۵۵ متر

**قیمت تقریبی:** $$ (متوسط)

---

### 📌 **Cat6a (Category 6 Augmented)**

**توضیحات:** نسخه پیشرفته Cat6 با پشتیبانی کامل از ۱۰ گیگابیت تا ۱۰۰ متر.

**ویژگی‌ها:**
- پشتیبانی کامل از ۱۰ گیگابیت در ۱۰۰ متر
- پهنای باند ۵۰۰ مگاهرتز (۲ برابر Cat6)
- کاهش بسیار زیاد تداخل (Alien Crosstalk)
- ضخیم‌تر و سنگین‌تر از Cat6
- معمولاً با شیلد (FTP/STP)

**کاربردها:**
- دیتاسنترهای حرفه‌ای
- شبکه‌های سازمانی بزرگ
- محیط‌های با تداخل بالا
- اتصالات ۱۰ گیگابیت در سراسر ساختمان

**مزایا:** عملکرد کامل ۱۰ گیگابیت، کاهش تداخل

**معایب:** ضخیم‌تر، سخت‌تر برای نصب، قیمت بالاتر

**قیمت تقریبی:** $$$ (بالا)

---

### 📌 **Cat7 (Category 7) - کلاس F**

**توضیحات:** استاندارد پیشرفته با شیلد کامل (S/FTP) برای محیط‌های با تداخل بالا.

**ویژگی‌ها:**
- شیلد کامل (هر جفت + کلی)
- پشتیبانی از ۱۰ گیگابیت در ۱۰۰ متر
- پهنای باند ۶۰۰ مگاهرتز
- کانکتور GG45 (سازگار با RJ45)
- بسیار مقاوم در برابر نویز

**کاربردها:**
- دیتاسنترهای بزرگ
- محیط‌های صنعتی با نویز بالا
- مراکز داده با تراکم بالا
- شبکه‌های با نیازمندی‌های خاص

**مزایا:** محافظت عالی، عملکرد پایدار

**معایب:** هزینه بالا، نیاز به کانکتور خاص، دشواری نصب

**قیمت تقریبی:** $$$$ (بسیار بالا)

---

### 📌 **Cat8 (Category 8)**

**توضیحات:** جدیدترین استاندارد برای دیتاسنترهای نسل جدید با سرعت‌های ۲۵ و ۴۰ گیگابیت.

**ویژگی‌ها:**
- پشتیبانی از ۲۵-۴۰ گیگابیت بر ثانیه
- فاصله محدود به ۳۰ متر (مناسب دیتاسنتر)
- پهنای باند ۲۰۰۰ مگاهرتز (۲ گیگاهرتز!)
- شیلد کامل (S/FTP)
- کانکتور RJ45 با استاندارد جدید

**کاربردها:**
- دیتاسنترهای نسل جدید
- اتصال سوئیچ‌های ۲۵/۴۰ گیگابیت
- شبکه‌های ذخیره‌سازی پرسرعت
- محیط‌های HPC (محاسبات با عملکرد بالا)

**مزایا:** بالاترین سرعت، پهنای باند بسیار بالا

**معایب:** فاصله محدود، گران‌قیمت، نیاز به تجهیزات خاص

**قیمت تقریبی:** $$$$$ (گران‌ترین)

# انواع شیلد در کابل‌های شبکه

## محافظت در برابر تداخل

کابل‌های شبکه بر اساس میزان محافظت در برابر تداخل الکترومغناطیسی (EMI) و تداخل بین جفت‌ها (Crosstalk) به انواع زیر تقسیم می‌شوند:

| نوع شیلد | نام کامل | توضیحات | کاربرد |
|---------|---------|---------|--------|
| **U/UTP** | Unshielded/Unshielded Twisted Pair | بدون هیچ شیلدی | محیط‌های معمولی، خانگی |
| **F/UTP** | Foiled/Unshielded Twisted Pair | شیلد فویل عمومی | محیط‌های با تداخل متوسط |
| **S/UTP** | Shielded/Unshielded Twisted Pair | شیلد بافته عمومی | محیط‌های با تداخل بالا |
| **U/FTP** | Unshielded/Foiled Twisted Pair | شیلد فویل برای هر جفت | کاهش تداخل بین جفت‌ها |
| **F/FTP** | Foiled/Foiled Twisted Pair | شیلد فویل عمومی + هر جفت | حفاظت کامل |
| **S/FTP** | Shielded/Foiled Twisted Pair | شیلد بافته عمومی + فویل هر جفت | بهترین حفاظت |
| **SF/UTP** | Shielded Foiled/Unshielded Twisted Pair | شیلد بافته + فویل عمومی | حفاظت بالا |

**توضیح اصطلاحات:**
- **U (Unshielded):** بدون شیلد
- **F (Foiled):** شیلد فویل (ورقه نازک آلومینیومی)
- **S (Shielded):** شیلد بافته (تارهای مسی)
- **TP (Twisted Pair):** زوج‌به‌هم تابیده

# کانکتورها و پچ‌کوردها

## اتصالات کابل‌های شبکه

### 🎯 **کانکتور RJ45**

**توضیحات:** استانداردترین کانکتور برای کابل‌های زوج‌به‌هم تابیده. دارای ۸ پین و ۸ تماس (8P8C) است.

**انواع:**
- **RJ45 معمولی:** برای Cat5e و Cat6
- **RJ45 Cat6a:** شیلددار، با کیفیت بالاتر
- **RJ45 Cat7 (GG45):** کانکتور خاص با قابلیت‌های پیشرفته
- **RJ45 Cat8:** با شیلد کامل و کیفیت بالا

**پین‌آوت (Wiring Standards):**

| پین | رنگ (T568A) | رنگ (T568B) | سیگنال |
|-----|-------------|-------------|--------|
| ۱ | سفید-سبز | سفید-نارنجی | TX+ (ارسال مثبت) |
| ۲ | سبز | نارنجی | TX- (ارسال منفی) |
| ۳ | سفید-نارنجی | سفید-سبز | RX+ (دریافت مثبت) |
| ۴ | آبی | آبی | - |
| ۵ | سفید-آبی | سفید-آبی | - |
| ۶ | نارنجی | سبز | RX- (دریافت منفی) |
| ۷ | سفید-قهوه‌ای | سفید-قهوه‌ای | - |
| ۸ | قهوه‌ای | قهوه‌ای | - |

### 🎯 **انواع پچ‌کوردها**

| نوع | توضیحات | کاربرد |
|-----|---------|--------|
| **Straight-through (مستقیم)** | پین‌ها به همان ترتیب | اتصال دستگاه‌های مختلف (کامپیوتر به سوئیچ) |
| **Crossover (متقاطع)** | پین‌های ارسال/دریافت عوض می‌شوند | اتصال دستگاه‌های مشابه (سوئیچ به سوئیچ) |
| **Rollover (معکوس)** | پین‌ها معکوس می‌شوند | اتصال به پورت کنسول |

**نکته:** امروزه اکثر دستگاه‌ها از Auto-MDI/MDX پشتیبانی می‌کنند و تفاوت Straight/Crossover خودکار تشخیص داده می‌شود.

# بخش دوم: کابل‌های فیبر نوری

## فناوری پیشرفته انتقال داده

کابل‌های فیبر نوری از نور برای انتقال داده استفاده می‌کنند و نسبت به کابل‌های مسی مزایای بسیاری دارند از جمله سرعت بسیار بالا، مسافت طولانی، امنیت بالا و مصونیت در برابر تداخل الکترومغناطیسی.

### ساختار کابل فیبر نوری

```
┌─────────────────────────────────────────────────────────┐
│  🧵 روکش خارجی (Outer Jacket)                         │
│  ┌───────────────────────────────────────────────────┐  │
│  │  🧵 نخ‌های مقاوم (Strength Members)              │  │
│  │  ┌─────────────────────────────────────────────┐  │  │
│  │  │  🧵 غلاف (Buffer/Coating)                  │  │  │
│  │  │  ┌───────────────────────────────────────┐  │  │  │
│  │  │  │  🪞 پوسته (Cladding)                  │  │  │  │
│  │  │  │  ┌─────────────────────────────────┐  │  │  │  │
│  │  │  │  │  💡 هسته (Core)                  │  │  │  │  │
│  │  │  │  │  (نور در اینجا منتقل می‌شود)     │  │  │  │  │
│  │  │  │  └─────────────────────────────────┘  │  │  │  │
│  │  │  └───────────────────────────────────────┘  │  │  │
│  │  └─────────────────────────────────────────────┘  │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

### اجزای فیبر نوری:

| جزء | توضیحات |
|-----|---------|
| **هسته (Core)** | مرکز فیبر، نور در آن منتقل می‌شود (شیشه یا پلاستیک) |
| **پوسته (Cladding)** | لایه دور هسته با ضریب شکست کمتر برای بازتاب نور |
| **پوشش (Coating/Buffer)** | محافظ مکانیکی فیبر |
| **نخ‌های مقاوم (Strength Members)** | برای افزایش مقاومت کششی |
| **روکش خارجی (Outer Jacket)** | محافظ نهایی کابل |

## انواع فیبر نوری

### ۱. **فیبر تک‌حالته (Single-Mode Fiber - SMF)**

**توضیحات:** فیبری با هسته بسیار نازک (۸-۱۰ میکرون) که فقط یک مسیر (حالت) نور را عبور می‌دهد.

**ویژگی‌ها:**
- هسته بسیار نازک (۹ میکرون)
- پهنای باند بسیار بالا
- مسافت بسیار طولانی (تا ۱۰۰+ کیلومتر)
- استفاده از لیزر (Laser) برای تولید نور
- استاندارد: OS1، OS2

**کاربردها:**
- اتصالات بین‌شهری و بین‌المللی
- شبکه‌های اپراتوری (Carrier Networks)
- اتصالات دیتاسنترهای دور
- شبکه‌های شهری (MAN)
- زیرساخت اینترنت

**مزایا:** مسافت بسیار طولانی، پهنای باند بالا، عملکرد عالی

**معایب:** هزینه بالا، نیاز به تجهیزات خاص (لیزر)، نصب دقیق‌تر

**قیمت تقریبی:** $$$$ (بالا)

---

### ۲. **فیبر چندحالته (Multi-Mode Fiber - MMF)**

**توضیحات:** فیبری با هسته بزرگتر (۵۰-۶۲.۵ میکرون) که چندین مسیر (حالت) نور را عبور می‌دهد.

**ویژگی‌ها:**
- هسته بزرگتر (۵۰ یا ۶۲.۵ میکرون)
- مسافت محدودتر (تا ۵۵۰ متر برای ۱۰ گیگابیت)
- استفاده از LED یا VCSEL برای تولید نور
- استانداردها: OM1، OM2، OM3، OM4، OM5

**انواع فیبر چندحالته:**

| استاندارد | هسته | پهنای باند | مسافت (۱۰GbE) | مسافت (۴۰GbE) | کاربرد |
|-----------|------|-----------|---------------|---------------|--------|
| **OM1** | ۶۲.۵/۱۲۵ | ۲۰۰ MHz·km | ۳۳ متر | - | قدیمی، کاربرد محدود |
| **OM2** | ۵۰/۱۲۵ | ۵۰۰ MHz·km | ۸۲ متر | - | شبکه‌های قدیمی |

**کاربردها:**
- دیتاسنترها (اتصالات درون ساختمان)
- شبکه‌های سازمانی (کامپوس)
- اتصالات کوتاه‌برد (تا ۵۵۰ متر)

**مزایا:** هزینه کمتر، نصب آسان‌تر، استفاده از تجهیزات ارزان‌تر

**معایب:** مسافت محدود، پهنای باند کمتر (نسبت به تک‌حالته)

**قیمت تقریبی:** $$-$$$ (متوسط)

# بخش سوم: کابل‌های کواکسیال

## فناوری قدیمی اما همچنان کاربردی

کابل‌های کواکسیال (Coaxial) از یک هادی مرکزی، عایق، شیلد بافته و روکش خارجی تشکیل شده‌اند. اگرچه امروزه کمتر در شبکه‌های کامپیوتری استفاده می‌شوند، اما هنوز در برخی کاربردها مانند تلویزیون کابلی، دوربین‌های مداربسته و اینترنت کابلی کاربرد دارند.

### ساختار کابل کواکسیال

```
┌─────────────────────────────────────────────────────────┐
│  🧵 روکش خارجی (Jacket) - PVC/PE                      │
│  ┌───────────────────────────────────────────────────┐  │
│  │  🛡️ شیلد بافته (Braided Shield) - مس/آلومینیوم │  │
│  │  ┌─────────────────────────────────────────────┐  │  │
│  │  │  🔹 لایه فویل (Foil Shield) - آلومینیومی   │  │  │
│  │  │  ┌───────────────────────────────────────┐  │  │  │
│  │  │  │  🧵 عایق دی‌الکتریک (Dielectric)     │  │  │  │
│  │  │  │  ┌─────────────────────────────────┐  │  │  │  │
│  │  │  │  │  ⚡ هادی مرکزی (Center Conductor)│  │  │  │  │
│  │  │  │  │  (مس خالص یا فولاد روکش‌دار)    │  │  │  │  │
│  │  │  │  └─────────────────────────────────┘  │  │  │  │
│  │  │  └───────────────────────────────────────┘  │  │  │
│  │  └─────────────────────────────────────────────┘  │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

### انواع کابل‌های کواکسیال:

| نوع | امپدانس | کاربرد | توضیحات |
|-----|---------|--------|---------|
| **RG-58** | ۵۰Ω | شبکه‌های قدیمی (10Base2) | نازک، کاربرد کم |
| **RG-59** | ۷۵Ω | ویدئو، دوربین مداربسته | برای سیگنال‌های ویدئو |
| **RG-6** | ۷۵Ω | تلویزیون کابلی، اینترنت | ضخیم‌تر، پهنای باند بیشتر |
| **RG-11** | ۷۵Ω | مسافت‌های طولانی | ضخیم‌ترین، افت کم |
| **RG-62** | ۹۳Ω | شبکه‌های ArcNet | قدیمی، کاربرد ندارد |

### کاربردهای امروزی کواکسیال:

- **تلویزیون کابلی (CATV):** انتقال کانال‌های تلویزیونی
- **اینترنت کابلی (Cable Internet):** اتصال مودم کابلی (DOCSIS)
- **دوربین‌های مداربسته آنالوگ:** انتقال تصویر
- **سیستم‌های صوتی:** میکروفون‌ها و بلندگوهای حرفه‌ای
- **آنتن‌های RF:** اتصال آنتن به گیرنده

# بخش چهارم: کابل‌های تخصصی

## کابل‌های با کاربردهای خاص

### ۱. **کابل‌های تخت (Flat Cables)**

**کاربرد:** کابل‌کشی زیر فرش، زیر موکت، در فضاهای محدود

**مزایا:** نازک، انعطاف‌پذیر، قابلیت عبور از زیر درب

**معایب:** مقاومت کمتر، کاربرد محدود

---

### ۲. **کابل‌های ضدآتش (Plenum Cables)**

**کاربرد:** کابل‌کشی در سقف‌های کاذب و فضاهای تهویه (Plenum)

**ویژگی‌ها:**
- روکش از مواد مقاوم در برابر آتش (FEP، PTFE)
- تولید دود کم در هنگام آتش‌سوزی
- استاندارد UL 910

**علامت‌گذاری:** روی کابل عبارت "CMP" یا "Plenum" درج می‌شود

**قیمت:** ۲-۳ برابر کابل معمولی

---

### ۳. **کابل‌های بیرونی (Outdoor/Waterproof Cables)**

**کاربرد:** کابل‌کشی در فضای باز، زیرزمینی، در معرض آب و هوا

**ویژگی‌ها:**
- روکش مقاوم در برابر UV (آفتاب)
- عایق ضدآب (معمولاً با ژل)
- مقاومت در برابر دماهای مختلف
- مقاومت مکانیکی بالا

**نکته:** استفاده از کابل داخلی در فضای باز باعث تخریب سریع می‌شود.

---

### ۴. **کابل‌های خودنگهدار (Aerial Cables)**

**کاربرد:** نصب روی تیرهای برق و مخابرات (هوایی)

**ویژگی‌ها:**
- دارای سیم فولادی برای تحمل وزن
- مقاوم در برابر باد و باران
- معمولاً مشکی رنگ با مقاومت UV بالا

---

### ۵. **کابل‌های شناور (Marine/Submarine Cables)**

**کاربرد:** ارتباطات زیردریایی، بین قاره‌ها

**ویژگی‌ها:**
- چندین لایه محافظ فلزی و پلاستیکی
- مقاومت در برابر فشار آب
- حفاظت در برابر کوسه‌ها (با فولاد یا کولار)
- حاوی تقویت‌کننده‌های سیگنال (Repeater) در فواصل مشخص

**طول:** تا ۲۰۰۰۰+ کیلومتر

# اشتباهات رایج در انتخاب کابل

## پرهیز از اشتباهات

### ❌ **اشتباه ۱: استفاده از Cat5e برای شبکه ۱۰ گیگابیت**
Cat5e فقط تا ۱ گیگابیت را پشتیبانی می‌کند. برای ۱۰ گیگابیت باید از Cat6a یا بالاتر استفاده کنید.

### ❌ **اشتباه ۲: استفاده از کابل داخلی در فضای باز**
کابل‌های داخلی در برابر UV، رطوبت و دما مقاوم نیستند و سریع تخریب می‌شوند.

### ❌ **اشتباه ۳: نادیده گرفتن فاصله استاندارد**
کابل‌های UTP حداکثر تا ۱۰۰ متر کار می‌کنند. بیشتر از آن باعث افت سیگنال می‌شود.

### ❌ **اشتباه ۴: کابل‌کشی نزدیک به کابل‌های برق**
تداخل الکترومغناطیسی از کابل‌های برق باعث اختلال در عملکرد شبکه می‌شود.

### ❌ **اشتباه ۵: عدم تست کابل پس از نصب**
حداقل ۲۰٪ کابل‌ها پس از نصب مشکل دارند. تست کامل ضروری است.

### ❌ **اشتباه ۶: استفاده از کابل‌های بی‌کیفیت**
کابل‌های تقلبی یا بی‌کیفیت (با مس آلومینیومی) عملکرد ضعیفی دارند.

### ❌ **اشتباه ۷: عدم توجه به شعاع خمش**
خم شدن بیش از حد کابل باعث شکستگی یا افت سیگنال می‌شود.

### ❌ **اشتباه ۸: برچسب‌گذاری نادرست**
عدم برچسب‌گذاری دقیق، عیب‌یابی را بسیار دشوار می‌کند.

# روندها و آینده کابل‌های شبکه

## تحولات آینده در فناوری کابل‌ها

### ۱. **افزایش سرعت**
- کابل‌های نسل جدید با سرعت‌های ۴۰، ۱۰۰ و ۴۰۰ گیگابیت
- استانداردهای جدید مانند Cat8.2 و فراتر

### ۲. **فیبر نوری به خانه (FTTH - Fiber to the Home)**
- جایگزینی کابل‌های مسی با فیبر نوری در شبکه‌های دسترسی
- سرعت‌های ۱، ۲ و ۱۰ گیگابیت برای کاربران خانگی
- کاهش هزینه فیبر نوری

### ۳. **کابل‌های هوشمند (Smart Cables)**
- با تراشه‌های تشخیص و عیب‌یابی داخلی
- گزارش خودکار وضعیت کابل

### ۴. **کابل‌های با توان بالا (PoE++)**
- انتقال توان بالاتر (تا ۱۰۰ وات) برای دستگاه‌های IoT
- کابل‌های با هادی ضخیم‌تر (AWG 22-23)

### ۵. **کابل‌های سازگار با محیط زیست**
- استفاده از مواد قابل بازیافت
- کاهش مصرف انرژی در تولید

### ۶. **کابل‌های کوانتومی**
- انتقال اطلاعات با استفاده از فوتون‌های درهم‌تنیده
- امنیت بسیار بالا
- هنوز در مرحله تحقیق

# سوالات متداول (FAQ)

**۱. تفاوت Cat5e و Cat6 چیست؟**
Cat6 پهنای باند بالاتر (۲۵۰ مگاهرتز در مقابل ۱۰۰)، سرعت بالاتر در فواصل کوتاه (۱۰ گیگابیت تا ۵۵ متر) و تداخل کمتر دارد.

**۲. آیا می‌توانم از کابل Cat7 با کانکتور RJ45 استفاده کنم؟**
بله، کابل‌های Cat7 با کانکتور GG45 عرضه می‌شوند که با RJ45 سازگار است، اما برای بهترین عملکرد به کانکتور مخصوص نیاز دارند.

**۳. چرا کابل فیبر نوری گران‌تر است؟**
به دلیل مواد (شیشه/پلاستیک با کیفیت بالا)، فرآیند تولید پیچیده، تجهیزات تخصصی نصب و کانکتورهای دقیق.

**۴. کدام کابل برای شبکه خانگی بهتر است؟**
Cat5e یا Cat6 برای شبکه خانگی کاملاً کافی است و قیمت مناسبی دارد.

**۵. تفاوت کابل Plenum و Riser چیست؟**
Plenum (CMP) برای سقف‌های کاذب با مقاومت بالا در برابر آتش و تولید دود کم. Riser (CMR) برای کابل‌کشی عمودی بین طبقات.

**۶. چگونه کابل تقلبی را تشخیص دهیم؟**
وزن کابل (کابل‌های تقلبی سبک‌ترند)، کیفیت برچسب، قیمت غیرمعمول، عدم وجود استاندارد (UL, ETL) و تست با دستگاه Fluke.

**۷. آیا کابل‌های مسی قدیمی (Cat5) قابل استفاده هستند؟**
برای سرعت‌های پایین (۱۰/۱۰۰ مگابیت) بله، اما برای شبکه‌های جدید (گیگابیت) توصیه نمی‌شوند.

**۸. حداکثر فاصله کابل فیبر نوری چقدر است؟**
بسته به نوع فیبر: چندحالته (تا ۵۵۰ متر برای ۱۰ گیگابیت) و تک‌حالته (تا ۱۰۰+ کیلومتر با تقویت‌کننده).

**۹. آیا کابل‌های شبکه برق را هم انتقال می‌دهند؟**
بله، با فناوری PoE (Power over Ethernet) که توان را از طریق کابل شبکه به دستگاه‌هایی مانند دوربین و تلفن IP منتقل می‌کند.

**۱۰. بهترین کابل برای دیتاسنتر چیست؟**
برای اتصالات درون رک: Cat8 (برای ۲۵/۴۰ گیگابیت) یا فیبر OM4/OM5. برای بین رک‌ها: Cat6a یا فیبر چندحالته.

**۱۱. آیا کابل‌های شبکه تاریخ انقضا دارند؟**
خیر، اما با پیشرفت فناوری، استانداردهای قدیمی‌تر پاسخگوی نیازهای جدید نیستند و نیاز به تعویض دارند.

**۱۲. تفاوت کابل UTP و STP چیست؟**
UTP بدون شیلد (ارزان، سبک) و STP با شیلد (مقاوم در برابر تداخل، گران‌تر، ضخیم‌تر).

# خلاصه و جمع‌بندی

انتخاب کابل شبکه مناسب، یکی از اساسی‌ترین تصمیمات در طراحی و پیاده‌سازی زیرساخت شبکه است. با درک صحیح از انواع کابل‌ها، استانداردها، کاربردها و معیارهای انتخاب، می‌توانید شبکه‌ای کارآمد، سریع و پایدار ایجاد کنید که نیازهای حال و آینده سازمان را برآورده سازد.

**نکات کلیدی:**

1. کابل‌های شبکه به دو دسته اصلی مسی (زوج‌به‌هم تابیده و کواکسیال) و فیبر نوری تقسیم می‌شوند

2. استانداردهای Cat5e تا Cat8 برای کابل‌های مسی با سرعت‌های متفاوت وجود دارند

3. Cat6a بهترین انتخاب برای شبکه‌های ۱۰ گیگابیت سازمانی است

4. فیبر نوری برای مسافت‌های طولانی و سرعت‌های بسیار بالا (۱۰۰+ گیگابیت) ضروری است

5. انتخاب کابل به نیاز سرعت، مسافت، محیط نصب و بودجه بستگی دارد

6. نصب صحیح و تست کابل‌ها برای عملکرد مطلوب حیاتی است

7. برای پروژه‌های جدید، حداقل از Cat6a استفاده کنید تا آینده‌نگری داشته باشید

8. در دیتاسنترها، ترکیب Cat8 و فیبر نوری بهترین عملکرد را ارائه می‌دهد

9. استانداردهای بین‌المللی (TIA/EIA، ISO/IEC) برای کابل‌کشی حرفه‌ای الزامی هستند

10. آینده به سمت فیبر نوری، سرعت‌های بالاتر و کابل‌های هوشمند حرکت می‌کند
